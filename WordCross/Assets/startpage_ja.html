WordCross

WordCrossは複数のオンライン辞書を簡単に切り替えながら検索できるアプリです。

How to Use

Just type any English word(s) into the search box (on the top left side) and press Enter. You'll get the definition in the dictionary listed on the top in the dictionary list (the left panel).

You can also look up phrases.

辞書の管理

追加

メニューバーの「辞書の追加」を選択し、以下の三つの情報を入力してください。

Name

辞書リストに表示される名前です。分かりやすいものをつけてください。

Base URL

Look up any word in the dictionary you are registering. Specify the part of the URL excluding your search word.

For example, when you look up "sleepy" in the Merriam Webster, the URL of the result page will be "https://www.merriam-webster.com/dictionary/sleepy". In this case, the base url is "https://www.merriam-webster.com/dictionary/".

Separator

It is necessary when you look up phrases. It specifies how dictionaries bind multiple words.

For example, when you look up "go to bed" in the Collins Thesaurus, the URL of the result page will be "https://www.collinsdictionary.com/dictionary/english-thesaurus/go-to-bed". The "go to bed" was converted into "go-to-bed", so the separator should be "-".

When the separator is unspecified, a whitespace will be used.

並べ替え

辞書リストでドラッグ＆ドロップするだけです！

削除

辞書を右クリックして「削除」を選択してください。

Other Functions

Adblock

Check "Disable Ads" in the menu bar to enable the adblock function. Beware that it is experimental, so if you encounter any issues, please uncheck it.
